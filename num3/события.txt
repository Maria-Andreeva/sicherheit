Поиск события успешного входа (ID 4624)
Subject: информацию о пользователе, который вошел в систему.
Logon Type: тип входа (например, удалённый, локальный или через сеть).
Account Name: имя учетной записи пользователя.
Workstation Name: имя компьютера, с которого был осуществлен вход (если применимо).
Повторный вход с неверным паролем (поиск события ID 4625)
Для события с ID 4625 (неуспешный вход) информация будет схожа, но также может содержать причину неудачной попытки входа.
Failure Information: причина неудачного входа (неверный пароль, заблокированная учетная запись и т.д.).
Помимо успешных (ID 4624) и неудачных (ID 4625) входов, в журнале безопасности могут быть и другие полезные события, связанные с безопасностью и входами в систему. Рассмотрите следующие ID:

ID 4647 — попытка выхода из системы.
ID 4634 — завершение сеанса (выход из системы).
ID 4672 — особые привилегии при входе в систему.
